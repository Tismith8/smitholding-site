# Smitholding LLC Website

This is a production-ready starter for Smitholding.com built with Next.js.

## Quick start

1. Install Node.js LTS
2. Open this folder in VS Code
3. Run:

```bash
npm install
npm run dev
```

4. Open `http://localhost:3000`

## Environment variables

Copy `.env.example` to `.env.local` and set:

```env
RESEND_API_KEY=re_your_api_key_here
CONTACT_FROM_EMAIL="Smitholding Website <onboarding@resend.dev>"
```

## Logo files

Replace these placeholders with your real logo files:

- `public/logo-icon.png`
- `public/logo-full.png`
